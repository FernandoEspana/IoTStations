IoTStations en una SPA(Single Page Aplication) creada para unificar multiples tecnologias, como Arduino usando sensores (Temperatura, Humedad, GPS) ademas de multiples librerias para hacer envio de datos entre sensores y placas de desarrollo. Se usa firebase siendo este un backend como servicio y finalmente Angular 8 como framework frontend, el cual permite conectar al backend una forma muy sencilla.

Elementos de hardware usados en el proyecto

  • Linkit One
  • NodeMCU
  • DHT-11
  • Dsiplay OLED 128x64 I2C
